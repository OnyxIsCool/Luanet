# Luanet

**Luanet** is a Lua script analysis and optimization toolkit designed to
help developers improve **code quality**, **performance**, and
**resource efficiency**. It provides automated insights and structured
recommendations to support cleaner and more maintainable Lua projects.

Luanet is built for developers who want greater visibility into script
behavior and execution flow.
